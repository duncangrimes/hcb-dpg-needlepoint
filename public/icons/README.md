# PWA Icons

Generate these icons from a source image (ideally 1024x1024 PNG with transparent background):

Required sizes:
- icon-72.png (72x72)
- icon-96.png (96x96)
- icon-128.png (128x128)
- icon-144.png (144x144)
- icon-152.png (152x152)
- icon-192.png (192x192) — primary, maskable
- icon-384.png (384x384)
- icon-512.png (512x512) — primary, maskable

Tools:
- https://realfavicongenerator.net/
- https://maskable.app/editor (for maskable icons)

The 192 and 512 icons should be "maskable" — meaning the important content is within the safe zone (inner 80% circle).
